# CAGED システム

## 概要

CAGEDシステムとは、ギターの開放弦コードフォーム（C・A・G・E・D）が
フレットボード上で連続してつながることを利用した指板理解の体系。
同じコードを5種類のフォームで弾き分けることができ、
スケールポジション・アルペジオ・ソロの「地図」になる。

---

## 5つのコードフォーム

### C フォーム（ルート: 2弦）
```
開放弦:  x 3 2 0 1 0
        (x A G C E C)
```
ルート音は2弦と5弦。
カポやバレーで移動可能。

### A フォーム（ルート: 5弦）
```
開放弦:  x 0 2 2 2 0
        (x A E A C E)
```
ルート音は5弦。バレーコード(Bm等)の基本。

### G フォーム（ルート: 6弦・1弦）
```
開放弦:  3 2 0 0 0 3
        (G B G D G B... G)
```
ルート音は6弦・1弦。大きなフォームで音域が広い。

### E フォーム（ルート: 6弦）
```
開放弦:  0 2 2 1 0 0
        (E B E G# B E)
```
ルート音は6弦。最もよく使われるバレーコードの基本形。
Fコード = Eフォームを1フレットバレー。

### D フォーム（ルート: 4弦）
```
開放弦:  x x 0 2 3 2
        (x x D A D F#)
```
ルート音は4弦。高音弦側に特化したフォーム。

---

## フレットボード上のつながり

Cメジャーコードを例にCAGEDが連続する様子：

```
フレット:  0    3    5    7    8    10   12
         |    |    |    |    |    |    |
Cフォーム:[C]──────────────────────────[C オクターブ上]
Aフォーム:     [C]
Gフォーム:          [C]
Eフォーム:               [C]
Dフォーム:                    [C]
```

各フォームのルート位置を押さえれば、ルートを変えても同じ形が使える。

---

## スケールポジションとの対応

マイナーペンタトニック（Aマイナーを例）の5ポジションがCAGEDと1対1対応する。

| ポジション | CAGEDフォーム | 主要フレット帯（Amルート）|
|-----------|-------------|----------------------|
| 1 (BOX1)  | E フォーム   | 5〜8 フレット |
| 2 (BOX2)  | D フォーム   | 7〜10 フレット |
| 3 (BOX3)  | C フォーム   | 9〜12 フレット |
| 4 (BOX4)  | A フォーム   | 12〜15 フレット |
| 5 (BOX5)  | G フォーム   | 2〜5 フレット |

このアプリの **Fret Mask** 機能でフレット範囲を絞ると、
特定のCAGEDポジションだけを練習できる。

### BOX1（Eフォーム / 5〜8フレット）

最もよく使われるポジション。
6弦5フレット(A)がルート。

```
6弦: ●  .  ●  .
5弦: .  ●  .  ●
4弦: .  ●  .  ●
3弦: ●  .  ●  .
2弦: ●  .  ●  .
1弦: ●  .  ●  .
    5f  6f  7f  8f
```

### BOX2（Dフォーム / 7〜10フレット）

```
6弦: .  ●  .  ●
5弦: ●  .  ●  .
4弦: ●  .  ●  .
3弦: .  ●  .  ●
2弦: .  ●  .  ●
1弦: .  ●  .  ●
    7f  8f  9f  10f
```

---

## CAGEDでルート音を探す

ギターでは同じ音が複数の弦・フレットに存在する。
ルートを変えるとき「どの弦のどのフレットがルートか」を CAGED で把握する。

| 弦 | ルートのフォーム |
|----|----------------|
| 6弦 | E フォーム / G フォーム |
| 5弦 | A フォーム / C フォーム |
| 4弦 | D フォーム |
| 2弦 | C フォーム |

---

## 実践的な使い方

1. **1つのポジションを完全に覚える** → BOX1（Eフォーム）から始める
2. **上下のポジションを連結する** → BOX1 ↔ BOX2 ↔ BOX3 の順でつなぐ
3. **コードトーンを意識する** → コードの3度・5度・7度がどこにあるか把握
4. **フレーズを横移動させる** → 同じフレーズを別フォームで弾く練習

このアプリで **Fret Mask** を使ってポジションを固定 → 慣れたら範囲を広げる、
という練習フローが CAGED 習得に効果的。
