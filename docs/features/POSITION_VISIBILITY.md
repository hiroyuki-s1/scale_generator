# ポジション表示/非表示機能 仕様書

## 1. 概要

同一ピッチ（異弦同音）のポジションを個別に表示/非表示できる機能。
例: Gメジャースケールの「1弦3フレット」と「6弦3フレット」は同じ音名だが、
任意のポジションだけを表示/非表示にできる。

---

## 2. 用語

- **異弦同音**: 異なる弦・フレットだが同じ音名（オクターブ違いを含む）のポジション
- **非表示ポジション**: ユーザーがタップして隠したポジション（薄く表示）

---

## 3. 機能要件

### 3.1 操作方法（スケールエディター）

- 指板上のドットを**長押し（または専用モードで通常タップ）** → 表示/非表示をトグル
- 非表示にしたポジションは**薄くフェード表示**（完全に消えない）
  - 薄く見せることで「非表示にしてある」ことをユーザーが認識できる
  - opacity: 0.25 程度

### 3.2 表示モード切り替え

```
指板上部にトグルボタン:
[ ポジション編集モード ON/OFF ]
```

- ON 時: ドットタップで表示/非表示トグル
- OFF 時: 通常操作（度数の選択等）

### 3.3 一括リセット

- 非表示ポジションを全て表示に戻す「リセット」ボタン

### 3.4 印刷時の扱い

- 非表示ポジションは印刷でも**表示されない**（viewBoxから除外）

---

## 4. データ構造

スケールの状態に `hiddenPositions` フィールドを追加。

```js
// 既存の scale オブジェクト
{
  rootIndex: 0,
  activeDegrees: new Set([0,2,4,7,9]),
  // ...existing fields...

  // 新規追加
  hiddenPositions: new Set([
    'g4s1',  // guitar: 4fret, string 1 → "g{fret}s{string}"
    'g9s3',
  ])
}
```

キー形式: `g{fret}s{string}` （fret=フレット番号, string=弦番号 0始まり）

---

## 5. 受け入れ条件

| # | 条件 |
|---|------|
| AC-01 | ポジション編集モードON時、ドットタップで非表示にできる |
| AC-02 | 非表示ポジションは薄く（opacity低下）表示される |
| AC-03 | もう一度タップすると再表示される |
| AC-04 | 「リセット」で全ポジションが表示状態に戻る |
| AC-05 | 非表示設定がlocalStorageに保存される |
| AC-06 | 印刷時に非表示ポジションは出力されない |
| AC-07 | ソングブック保存/読み込み時に非表示設定も保持される |
